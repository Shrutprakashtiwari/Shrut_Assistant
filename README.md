# Shrut Assistant 🤖

AI-powered voice assistant built in Python with speech recognition and task automation.

## 🚀 Features
- Voice command recognition
- Open applications
- Tell time
- Command-based automation

## 🛠️ Tech Stack
- Python
- SpeechRecognition
- pyttsx3
- subprocess

## ▶️ How to Run
```bash
pip install -r requirements.txt
python assistant.py
